# Role: Field Employee / Contractor

Field users focus on rapid reporting, task execution, and personal compliance via the mobile app.

**Total Pages Accessible: 8 (Primarily Mobile)**

---

## 1. Mobile Home / My Tasks (Mobile App)
*   **Purpose:** View assigned work and priority alerts.
*   **Content:** Personalized task list (Audits, CAPAs), Unread notifications.
*   **Primary Actions:** Start task, View notification.

## 2. Incident / Near Miss Quick Report (Mobile App)
*   **Purpose:** Report a safety event in under 30 seconds.
*   **Content:** Camera interface, Location (GPS/Dropdown), Type, Description.
*   **Primary Actions:** Take photo, Submit report.

## 3. Hazard Observation Report (Mobile App)
*   **Purpose:** Report unsafe conditions or behaviors.
*   **Content:** Severity selector, Photo capture, Corrective action text.
*   **Primary Actions:** Submit hazard.

## 4. Permit Request Form (Mobile App)
*   **Purpose:** Request a field permit for high-risk work.
*   **Content:** Task selector, Asset/Zone, Controls checklist (RA/SOP).
*   **Primary Actions:** Sign and Submit request.

## 5. SOP Search & Detail (Mobile App)
*   **Purpose:** Access safety instructions in the field.
*   **Content:** Search bar, PDF viewer, Acknowledgement button.
*   **Primary Actions:** Acknowledge SOP.

## 6. Audit Checklist Execution (Mobile App)
*   **Purpose:** Conduct field inspections and audits.
*   **Content:** Question list (Yes/No/NA), Comment boxes, Photo evidence capture.
*   **Primary Actions:** Answer question, Complete audit.

## 7. My CAPA Actions (Mobile App)
*   **Purpose:** Close corrective actions assigned to self.
*   **Content:** Action description, Evidence upload (Photo/Doc).
*   **Primary Actions:** Submit closure evidence.

## 8. AI Safety Advisor Chat (Mobile App)
*   **Purpose:** Ask safety-related questions.
*   **Content:** Chat interface with cited SOP/Policy answers.
*   **Primary Actions:** Ask question.
