# Interaction Guide: Executive Sponsor (Strategy Dashboard)

This guide details the strategic oversight interactions for Executive leadership.

---

## Screen 1: Executive Safety Intelligence Dashboard
**High-level performance monitoring.**

### UI Elements & Interhow to connect tactions
| Element | UI Type | Action | Destination / Result |
| :--- | :--- | :--- | :--- |
| **Safety Score Gauge**| Chart | Hover | Shows score breakdown (Leading vs. Lagging indicators) |
| **Regional Selector** | Map/Dropdown | Click | Filters all data to a specific country or region |
| **"Predictive Risk"** | AI Card | Click | Navigates to `AI Safety Intelligence Dashboard` |
| **"Export Board Pack"**| Action Button | Click | Generates a 10-page PDF report for Board meetings |

---

## Screen 2: AI Safety Intelligence Dashboard
**Forward-looking risk and automated insights.**

### UI Elements & Interactions
| Element | UI Type | Action | Destination / Result |
| :--- | :--- | :--- | :--- |
| **Risk Trend Line** | Chart | Scrub | Reviews risk forecast for next 7, 14, or 30 days |
| **"Weekly Briefing"** | Document Preview| Click | Opens full text of AI-generated safety briefing |
| **"Approve Briefing"** | Primary Button | Click | Finalizes briefing and triggers distribution to site managers |
| **Citation Link** | Underlined Text | Click | Opens the original Policy or SOP cited by the AI |

---

## Screen 3: Compliance Scorecard
**Monitoring organizational standing and ISO health.**

### UI Elements & Interactions
| Element | UI Type | Action | Destination / Result |
| :--- | :--- | :--- | :--- |
| **ISO Standard Tabs** | Tabs | Click | Switches view between 45001 (Safety) and 14001 (Env) |
| **Gap Analysis Bar** | Chart | Click | Drills down into specific missing ISO clauses |
| **"Export Evidence"** | Button | Click | Generates a zip folder of audit evidence for external auditors |
